# WasherLord - 智能洗衣机监控助手

## 项目简介

WasherLord 是一个基于 Flutter 框架开发的智能洗衣机监控应用，帮助你在xxx洗衣房场景中实时监控洗衣机状态，自动抢占可用机器，并简化洗衣流程。

## 核心功能

### 🚀 实时洗衣机监控

- 显示洗衣房所有洗衣机状态（可用、漂洗中、洗涤中、已占用）
- 直观展示每台洗衣机的剩余时间
- 支持开始/停止监控操作

### ⚡ 智能抢占系统

- 自动检测可用洗衣机
- 一键创建洗衣订单
- 订单信息实时追踪（订单号、支付状态）

### ⧉ 个性化配置中心

- **洗衣机排除配置**：自定义忽略特定洗衣机
- **洗衣模式预设**：
  - 快洗（23分钟）¥3.5
  - 常规（40分钟）¥3.7
  - 超长洗（45分钟）¥4.2
  - 脱水（6分钟）¥1.0
- **支付配置**：开启自动支付功能

### 👤 用户账户管理

- 多账号切换支持
- 用户配置导入/导出

## 技术架构

- **框架**：Flutter (跨平台支持iOS/Android)
- **本地存储**：SharedPreferences

## 使用指南

1. **登录账号**：输入手机号使用当前账号
2. **配置偏好**：
   - 设置要排除的洗衣机
   - 选择常用洗衣模式
   - 开启自动支付功能
3. **开始监控**：点击"开始监控"按钮
4. **抢占洗衣机**：当检测到可用机器时自动创建订单
5. **完成支付**：及时支付订单开始洗衣
